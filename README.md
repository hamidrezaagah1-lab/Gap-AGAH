# سامانه وب تحلیل حقیقت‌جو (نسخه ۶۶)

این وب‌اپلیکیشن بر پایه فریم‌ورک تحلیلی عدالت‌محور، با اولویت‌بخشی به کرامت انسانی و حق‌طلبی توسعه یافته است.

## ویژگی‌ها
1. **رابط کاربری کاملاً فارسی و راست‌چین (`dir="rtl"`)**
2. **پایداری تاریخچه گفتگو**: انتقال داده‌ها از `session` مرورگر به پایگاه‌داده محلی SQLite.
3. **تست‌های واحد مجزا**: پوشش تست برای ۴ حالت تفکر انتقادی و تحلیلی.

## راهنمای اجرا
1. نصب پیش‌نیازها:
   ```bash
   pip install -r requirements.txt
   ```
2. اجرای وب‌اپلیکیشن:
   ```bash
   python web_app.py
   ```
3. اجرای تست‌ها:
   ```bash
   python -m unittest tests/test_framework.py
   ```
